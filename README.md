# User Funnel Analysis Dashboard

## Overview

This project analyzes user progression through a product funnel.

The dashboard provides:

- Funnel Visualization
- Conversion Rate Analysis
- Drop-off Detection
- Segment Comparison
- Time-to-Convert
- Data Quality Report
- Automated Recommendations

## Tech Stack

- Python
- Pandas
- Streamlit
- Plotly

## Install

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```

Dataset

```
data/funnel_events_sample.csv
```
